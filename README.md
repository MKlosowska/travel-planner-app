# Travel Planner App

## Technologie

- C#
- .NET 8
- Blazor Server
- ASP.NET Core Web API / Minimal API
- Entity Framework Core
- SQLite
- HTML/CSS

## Funkcjonalności w projekcie

- logowanie e-mail + hasło,
- blokada konta po 5 błędnych próbach na 15 minut,
- reset hasła z kodem demo,
- wylogowanie i ochrona widoków,
- dodawanie, edycja i usuwanie podróży,
- budżet podróży,
- dodawanie wydatków i przeliczanie wykorzystanego budżetu,
- upload dokumentów PDF z walidacją formatu i limitu 10 MB,
- podgląd i pobieranie PDF przez endpoint API,
- upload zdjęć i galeria miniatur,
- limit 500 MB plików na jedną podróż,
- lokalizacje / punkty na mapie przez formularz współrzędnych i tagi,
- lista pakowania z checkboxami, przekreślaniem i paskiem postępu,
- eksport listy pakowania do PDF,
- współużytkownicy podróży po e-mailu,
- ustawienie włącz/wyłącz powiadomienia.
